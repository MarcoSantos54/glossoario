Glossário de Desenvolvimento Web

Grupo 1: Fundamentos da Web

HTML
Linguagem de marcação usada para estruturar conteúdo na web, definindo elementos como títulos, parágrafos e imagens.
HyperText Markup Language Linguagem de Marcação de Hipertexto Estrutura de Página Web
JavaScript
Linguagem de programação interpretada, usada para adicionar interatividade e lógica dinâmica em páginas web.
JS ECMAScript Linguagem de Script
Desenvolvendo
Processo de criar ou construir software, envolvendo planejamento, codificação e testes.
Criando Programando Construindo
Código
Sequência de instruções escritas em uma linguagem de programação para resolver problemas ou automatizar tarefas.
Script Programa Fonte
Comentários
Textos não executáveis inseridos no código para explicar sua funcionalidade, facilitando manutenção e colaboração.
Anotações Notas Observações

Grupo 2: Lógica e Execução

Função
Bloco de código reutilizável que realiza uma tarefa específica e pode ser chamado múltiplas vezes.
Rotina Procedimento Sub-rotina
Parâmetro
Variável definida na declaração de uma função para receber valores ao ser chamada.
Argumento Variável de Entrada Entrada
Calculado
Resultado obtido por meio de operações matemáticas ou lógicas em um programa.
Computado Processado Determinado
Verifica
Processo de checar condições ou validade de dados para garantir correção.
Checa Valida Confere
Tempo
Medida de duração ou intervalo usado em programação para delays, cronômetros ou otimizações.
Duração Intervalo Período

Grupo 3: Manipulação de Dados

Receber
Ação de obter dados de uma fonte externa, como input do usuário ou API.
Obter Aceitar Capturar
Incrementar
Aumentar o valor de uma variável por uma unidade ou quantidade específica.
Aumentar Acrescentar Elevar
Atualizar
Modificar dados ou estados existentes para refletir mudanças recentes.
Renovar Refrescar Modificar
Retornar
Devolver um valor ou resultado de uma função para o chamador.
Devolver Enviar de Volta Responder
Agrupar
Organizar elementos em coleções ou estruturas para facilitar manipulação.
Agrupar Juntar Organizar
